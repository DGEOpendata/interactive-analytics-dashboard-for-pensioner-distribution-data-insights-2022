# Interactive Pensioners Distribution Dashboard

This repository provides the code and instructions to deploy an interactive analytics dashboard for the 'Pensioners Distribution Data for 2022' dataset. The dashboard is designed to provide visualizations and insights into the distribution of pensioners based on gender and quarterly data.

## Features

- Interactive visualizations (bar charts, line graphs, pie charts, etc.)
- Filter and analyze data by year, quarter, and gender
- Responsive design for desktop, tablet, and mobile devices
- Online hosting for easy access

## Requirements

To run this project locally, you will need:

- Python 3.7+
- Flask
- pandas
- plotly

## Installation

1. Clone the repository:
   bash
   git clone https://github.com/your-repo/pensioners-dashboard.git
   

2. Navigate to the project directory:
   bash
   cd pensioners-dashboard
   

3. Install the required Python libraries:
   bash
   pip install -r requirements.txt
   

## Dataset

Download the dataset files from the provided source and place them in the project directory. Ensure the file is named `Distribution_of_Pensioners_2022.xlsx`.

## Running the Application

1. Run the Flask application:
   bash
   python app.py
   

2. Open your web browser and visit `http://127.0.0.1:5000` to access the dashboard.

## Usage

- Use the filters to explore data by specific quarters or gender.
- Hover over the charts to view detailed information.
- Use the search functionality to find specific datasets or parameters.

## Future Enhancements

- Add more visualization types, such as heatmaps and scatter plots.
